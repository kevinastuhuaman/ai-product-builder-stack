# AI Product Builder Stack

An interactive, evidence-backed map of the models, infrastructure, browser agents, analytics, voice systems, and product tools I have used to ship real products.

This is not a logo wall. Every entry includes a lifecycle state, a product outcome, implementation evidence, and the projects where I used it.

## What it demonstrates

- AI product judgment across model routing, retrieval, agents, and human approval.
- Technical fluency across Azure, AWS, Terraform, PostgreSQL, search, and observability.
- Hands-on delivery across web, iOS, macOS, CLI, MCP, voice, and browser automation.
- A clear distinction between shipped, active, prototype, and historical experience.
- Public-safe communication that protects employer IP, user data, and infrastructure secrets.

## Interactive views

- **Stack:** filter by project, lifecycle state, tool, outcome, or evidence.
- **Systems:** inspect end-to-end flows for job discovery, browser agents, voice onboarding, and conversation follow-up.
- **Evidence detail:** open any entry to see the exact implementation or operating proof.
- **Machine-readable:** use [`stack.json`](https://kevinastuhuaman.github.io/ai-product-builder-stack/stack.json) or [`llms.txt`](https://kevinastuhuaman.github.io/ai-product-builder-stack/llms.txt).

## Run locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:4321/ai-product-builder-stack/`.

## Verify

```bash
npm test
```

The suite builds the static site and tests desktop and mobile behavior, filters, evidence dialogs, structured data, accessibility, and viewport overflow.
